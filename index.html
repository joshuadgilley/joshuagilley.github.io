Work

MARS MapMaker

Demo

MARS MapMaker is an application that was created to work in tandem with MARS from CIRDLES. The purpose of this MapMaker is to allow entitites to properly register their rock samples with SESAR (System for Earth Sample Registration) to aquire their IGSN (International Geo Sample Number). It's general purpose is to allow the user to submit an example file that contains a portion of their own data and it's metadata and to interact with the user to output a javascript mapping file that allows a mapping between their metadata and SESAR's metadata. This project is currently being tested by Columbia University.

Squid Web Service

Squid 3.0 is the replace for Squid 2.5. The goal was to provide full workflow support for processing and interpreting Shrimp data outputs in the form of Prawn xml files and ‘OP’ files including project, data, task, expression, visualizations, report, and parameters management. We were responsible for turning this application into a web-service using the MERN stack.
